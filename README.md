# 🔐 Safe Input Logger

A Python-based logging tool that records user input in a controlled and ethical environment. This project was developed as part of my cybersecurity internship to understand logging mechanisms while maintaining user privacy and security principles.

---

## ⚠️ Ethical Note

This project does **NOT** implement a system-wide keylogger.

Instead, it demonstrates logging functionality by capturing **user-provided input within the application only**. This ensures the project remains ethical, safe, and compliant with cybersecurity best practices.

---

## 🚀 Features

- Logs user input in a controlled environment
- Stores data in a file with timestamps
- Simple and easy-to-use CLI interface
- Demonstrates secure logging concepts without violating privacy

---

## 🧠 Concepts Covered

- Input Handling in Python
- File Operations (Read/Write)
- Timestamp Logging
- Ethical Considerations in Cybersecurity
- Responsible Software Development

---

## 🛠️ Tech Stack

- Python 3
- Standard Libraries (`datetime`)
